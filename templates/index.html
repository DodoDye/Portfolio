Avez-vous remarqué que ce site ?

  • Est accessible en HTTPS seulement, avec un certificat valide
  • Possède un nom de domaine, comme tous les sites à volonté publique
  • Laisse la possibilité de créer un compte et de se connecter

Ce que vous ne pouvez pas voir

  • Il est hébergé derrière une box privée
  • Cette page est servie par un serveur tournant sur un Raspberry Pi 3
  • Il est développé sous Node JS

Pourquoi présenter tout ceci ?

  • Concentrer mes principales compétences professionnelles en un seul projet
  • Mettre en pratique une formation récente sur Node JS
  • Faire original, pourquoi pas ?
